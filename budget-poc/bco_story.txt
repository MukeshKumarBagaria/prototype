User Story:
	•	As a BCO Creator, I want to submit the budget estimation for Expenditure for Verification.
	•	As a BCO Verifier, I want to review, edit and forward the Budget estimation for approval.
	•	As a BCO Approver, I want to review, edit and approve the budget estimation.
Pre-Conditions
	•	Budget Line Items to be mapped with BCO.
	•	Department-wise ceilings defined and approved by FD.
	•	Budget module activated by Finance Department.
	•	Budget Submission date within the time limit as per Budget Circular.
Process Flow
	•	Data Synchronization:
Based on Employee login, BCO linked with department details are fetched from HRMS through secure API.
	•	Auto-Fetch Components: System retrieves all budget line items assigned to BCO.
System retrieves actual Expenditure for budget line items occurred during previous 5 years.
System retrieves the Budget Allocation of previous year.
	•	BCO Creator:
BCO Creator receive file from multiple DDO for verification and compilation. OR
BCO Creator can also create Budget Estimates by himself. To create Budget estimate He selects the budget line item and enters the budget estimation for next 3 years and revised estimation for current year for Expenditure and Expenditure separately.
BCO Creator enters the data for each budget line item and saves.
BCO creator can create and enter token amount for new budget line for approval by Finance department (Assembly). 
BCO Creator submits to verifier once all budget line data is ready
	•	Rule Engine Validation:
Budget Estimate submitted should not be more that overall ceiling limit for Expenditure. If limit is exceeded display error message
	•	BCO Verifier:
BCO Verifier views the budget estimation submitted by BCO creator. BCO Verifier can either edit or return to BCO Creator or forward the file to BCO Approver.
BCO verifier can also create new budget line and enter token amount for approval by Finance department.


	•	BCO Approver:
BCO Approver views the budget estimation forwarded by BCO verifier. He can either edit or return to BCO Verifier or approve the file. 
BCO verifier can also create new budget line for Expenditure and enter token amount for approval by Finance department.
The approved budget will be sent to Admin HOA
	•	Audit Trail and Reporting: Each action creation, modification, verification, approval, rejection logged for audit.
